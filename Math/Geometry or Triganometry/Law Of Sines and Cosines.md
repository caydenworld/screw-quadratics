## Cosines
The law of cosines is a bit harder. The law of cosines states that $c^2=a^2+b^2-2ab\cos{C}$ 

The values are as follows:
```
                b
----------------------------
\  A                    C  /
 \                        /
  \                      /
   \                    /
    \                  /
     \                /
      \              /
       \            /
    c   \          /
         \        /     a
          \      /
           \  B /
            \  /
             \/
```

An example is below:

```
                x
----------------------------
\                          /
 \                        /
  \                      /
   \                    /
    \                  /
     \                /
      \              /
       \            /
   15   \          /
         \        /     14
          \ 119  /
           \    /
            \  /
             \/
```

$x^2=15^2+14^2-2(15)(14)\cos{119}$
